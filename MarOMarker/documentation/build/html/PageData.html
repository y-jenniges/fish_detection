Signal emitted when the experiment ID is changed.

Signal emitted when the image directory is changed.

Signal emitted when the image prefix is changed.

onBrowseImageDir()¶

Function that opens a dialog for the user to select a directory where the images are

onBrowseOutDir()¶

Function that opens a dialog for the user to select an output directory.

onCalcLength()¶

Gets current image list from photo viewer and calculates the length of animals whose coordinates are valid.

onCalenderSelectionChanged()¶

Function to handle a change of the calender widget selection.

onCameraConfigChanged(config_path)¶

Loads a new camera configuration from a json file path.

onCheckMatch()¶

Directs to LR screen (showing left and right images) if at least one match is already calculated.

onCheckPredictions()¶

Handles the click on ‘Check predictions on home screen’ button by navigating to the home screen.

onExpIdEditChanged()¶

Function to handle when the user changes the experiment id line edit

onImageDirEditChanged(text=None, updateVisuals=True)¶

Function to handle when the user changes the img_dir line edit

onNnActivated()¶

Handles the activation of the “run neural network” button. It gets the current image list from the photo viewer that it applies the neural network to (in a separate thread). It appends the new data to the data model.

onNnFinished()¶

Called when the predictions are done. Enables the data page and updates data table.

onNnProgress(i)¶

Called to report on the prediction progress. Updates the label displaying the number of predicted images and the progress bar.

onOutDirChanged()¶

Function to handle when the user changes the output dir line edit. It makes sure that a result file exists and creates one otherwise.

onPrefixEditChanged()¶

Function to handle when the user changes the image prefix line edit

onRectifyMatch()¶

Gets current image list from photo viewer, rectifies them and determines the positions of the respective animal on the right image using block matching.

onRectifyMatchFinished()¶

Called when the rectification and matching calculations are done. Enables the data page and triggers length calculations.

onRectifyMatchProgress(i)¶

Called to report on the rectification and matching progress. Updates the label displaying the number of rectified and matched images and the progress bar.

outputDirectoryChanged¶

Signal emitted when the output directory is changed.

restoreValues(settings)¶

Restores settings of page data from last program start

saveCurrentValues(settings)¶

Saves current settings on data page for next program start

updateImageDir()¶

Function to update the image directory according to the input from the calender widget

updateNumImages()¶

Function to check how many images are in the current image directory with the current prefix for the current date and updates the display of this count
