Photo Viewer¶

Class to display and switch between photos.

PhotoViewer class¶

class PhotoViewer.PhotoViewer(models, imageDirectory, imagePrefix, outputDir='', parent=None)¶

Bases: PyQt5.QtWidgets.QWidget

A photo viewer that contains a QGraphicsView to display the photos and draw the animals on.

Variables

  • image_directory (string) – Directory where the images are stored.

  • image_prefix (string) – Images will be displayed only if they start with this prefix.

  • output_dir (string) – Directory where the output file is stored.

  • cur_image_index (int) – Index of the current image in image_list.

  • image_list ([list<string>, list<string>]) – List containing all image pathes that have the given prefix, are of the selected date. First entry contains all left image pathes and second entry all right image pathes.
