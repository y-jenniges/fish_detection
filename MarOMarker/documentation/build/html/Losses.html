Neural Network Losses¶

Contains custom losses utilized during the training of the networks.

Losses.weighted_categorical_crossentropy(weights)¶

A weighted version of keras.objectives.categorical_crossentropy

Usage:

weights = np.array([0.5,2,10]) # Class one at 0.5, class 2 twice the normal weights, class 3 10x. loss = weighted_categorical_crossentropy(weights) model.compile(loss=loss,optimizer=’adam’)

Taken from:

https://gist.github.com/wassname/ce364fddfc8a025bfab4348cf5de852d (last access: 08.10.2020)

Parameters

weights (ndarray) – The numpy array has shape (C,) where C is the number of classes. It contains one weight per class.

Losses.weighted_mean_squared_error(y_true, y_pred)¶

MSE loss for heads and tail vector fields. All arrows that are not at the head or tail are ignored in this loss
